## 📌 Objective
Clean and analyze a Superstore-style sales dataset and build an 
interactive Excel dashboard to visualize:
- ✅ Total Sales by Category
- ✅ Profit by Category  
- ✅ Sales Trend Over Time (Monthly)
- ✅ Conditional Formatting to highlight high/low performers

---

## 🛠️ Tools Used
- Microsoft Excel
- Pivot-style SUMIF / SUMPRODUCT formulas
- Excel Charts (Bar Chart, Line Chart)
- Conditional Formatting (Color Scales + Rule-based)
- Excel Tables (structured references)
